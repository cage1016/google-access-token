google-access-token
===================

Easily generate a Google access token, optionally with a refresh token.

### Requirements

- Node.js + npm

### Installation

```sh
$ git clone git@github.com:nporteschaikin/google-access-token
$ cd google-access-token
$ npm install
```

### Usage

```sh
$ npm start
```
